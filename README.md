# HITLER JUOMAPELI

Selaimessa toimiva korttijuomapeli. Pelaajat nostavat vuorotellen kortin
sähköisestä pakasta (52 korttia, ei jokereita) ja tekevät mitä kortti käskee.
Toimii puhelimella ja tietokoneella, ei vaadi asennusta eikä nettiyhteyttä.

## Pelaaminen

Avaa `index.html` selaimessa tai julkaistu osoite, ja paina **Nosta kortti**.
Vuorot kiertävät pöydän ympäri. Sääntö-, Huora- ja Kysymysmestari-kortit
pysyvät voimassa, kunnes sama kortti nostetaan uudelleen.

## Julkaisu (GitHub Pages)

1. Luo uusi repo ja lataa tämä `index.html` (ja halutessasi tämä README).
2. **Settings → Pages → Branch: `main` / `(root)` → Save**.
3. Noin minuutin kuluttua peli on osoitteessa
   `https://kayttajanimesi.github.io/repon-nimi/`.

## Huom

Tarkoitettu täysi-ikäisille (18+). Kenenkään ei ole pakko juoda — vettä tai
limua voi käyttää vaihtoehtona.
